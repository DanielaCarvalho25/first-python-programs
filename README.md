# Meus Primeiros Programas em Python

Esse repositório foi criado para guardar meus primeiros exercícios de lógica de programação.  
Aqui estão atividades feitas no Google Colab, com foco em aprender o básico de Python e resolver problemas do dia a dia. Com programas simples que trabalham lógica, matemática e algumas simulações.


#Finanças e Comércio

- Cálculo de Troco: Calcula o valor que deve ser devolvido ao cliente.
- Salário com Comissão: Calcula o salário final com base nas vendas.
- Restaurante a Quilo: Calcula o valor da refeição com base no peso.
- Gasolina no Tanque: Calcula quantos litros podem ser abastecidos com um valor.
- Preço de Produto por Categoria: Calcula o preço de venda com base no tipo (Perecível ou Não Perecível), aplicando lucro diferente.


#Matemática e Lógica

- Par ou Ímpar: Verifica se um número é par ou ímpar.
- Número entre 20 e 50: Verifica se está dentro de um intervalo.
- Ordenação de Números: Mostra três números em ordem crescente.
- Quociente e Resto: Faz divisão inteira e mostra o resto.
- Média Aritmética: Calcula a média de notas.
- Antecessor e Sucessor: Mostra o número anterior e o próximo.
- Percentual de Votos: Calcula porcentagens de votos.


#Validações

- Verificação de Senha: Confere se a senha digitada está correta.

#Simulações

- Conta de Telefone: Calcula o valor da conta com base no tipo de cliente e minutos usados.

- Conta de Hotel: Calcula o valor da hospedagem com base no tipo de quarto, diárias e consumo, com descontos.


#Informações e Saúde

- Cálculo de IMC: Calcula o índice de massa corporal.
- Idade Atual: Calcula a idade com base no ano de nascimento.
- Conversão de Horas: Converte horas para minutos e segundos.

 #Tecnologias e Ferramentas
  Linguagem:  Python 3
  Ambiente: Google Colab
  Conceitos aplicados:  Variáveis, operadores aritméticos, entrada/saída de dados e formatação de strings (f-strings).

  O que estou aprendendo:
- Variáveis  
- Entrada e saída de dados  
- Operadores matemáticos  
- Condições (if/else)
