# 🐍 Meus Primeiros Programas de Lógica e Matemática em Python

Repositório criado para organizar meus estudos iniciais de programação. Aqui estão diversos exercícios de lógica desenvolvidos no **Google Colab**, focados em resolver problemas matemáticos e situações do cotidiano.

## 📑 O que este projeto resolve?
O arquivo principal contém scripts para os seguintes cálculos:

### 💰 Finanças e Comércio
* **Cálculo de Troco:** Informa o valor exato a ser devolvido ao cliente.
* **Salário com Comissão:** Calcula o bônus por vendas e o salário final do vendedor.
* **Restaurante a Quilo:** Calcula o valor a pagar com base no peso em gramas.
* **Gasolina no Tanque:** Define quantos litros podem ser abastecidos com um determinado valor.

### 🔢 Matemática e Lógica
* **Quociente e Resto:** Realiza a divisão inteira e mostra o que sobra.
* **Média Artimética:** Calcula a média de notas de alunos.
* **Antecessor e Sucessor:** Identifica os números vizinhos de um valor inteiro.
* **Percentual de Votos:** Calcula a porcentagem de votos brancos, nulos e válidos de uma eleição.

### 👤 Informações e Saúde
* **Cálculo do IMC:** Calcula o Índice de Massa Corporal com base em peso e altura.
* **Idade Atual:** Calcula a idade da pessoa a partir do ano de nascimento.
* **Conversão de Horas:** Transforma horas em minutos e segundos.

## 🛠️ Tecnologias e Ferramentas
* **Linguagem:** Python 3
* **Ambiente:** Google Colab
* **Conceitos aplicados:** Variáveis, operadores aritméticos, entrada/saída de dados e formatação de strings (f-strings).
